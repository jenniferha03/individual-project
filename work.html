Tribble Management System

Programming Language: C++.

The project involves creating a comprehensive software system to manage various aspects of a business that deals with Rainbow Tribbles. The system covers functionalities such as customer management, sales, order processing, and a customer portal for online interactions. The software also employs basic security measures, such as password encryption, to ensure user authentication.

Arduino Circuits

Programming Language: Arduino.

The first program simulates a 1-bit Arithmetic Logic Unit (ALU). The ALU is a fundamental component in digital circuits, capable of performing basic arithmetic and logic operations.

The second program simulates a 2-bit Full Adder. A Full Adder adds two binary numbers along with a carry input.
